## 目录结构

##### src:

###### --assets  存放静态资源

###### --components 视图相关组件

###### --network 网络请求

###### --plugins 存放插件(element-ui)

###### --router 路由

###### --store vuex

###### --utils 自定义工具

###### --views 视图



## 安装运行

```shell
克隆
$ git clone git@github.com:/P-J27/vue-music.git
或者
$ git clone git@gitee.com:p_pj/vue-music.git

切到根目录
$ cd front
安装依赖
$ npm install
运行
$ npm run serve
```
